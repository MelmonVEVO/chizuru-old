# Chizuru
Hoo boy.

This thing is designed to run in a Docker container. To do that, run these:
```shell
docker build -t chizuru .
docker run
```

If this thing is located on Hex (the uni gpu cloud), run these:
```shell
hare build -t <username>/chizuru .
hare run
```
